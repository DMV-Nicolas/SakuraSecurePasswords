Enero 2021 (version 1.0 Beta)

Comienzos de año se lanza Sakura Secure Passwords con el proposito de ser un administrador de contraseñas que se maneja desde la web, facil y sencillo de utilizar, este primer lanzamiento trae:


  • Implementacion de cuentas:

    ¡Registro e inicio de sesion!, ademas de requerimientos.

  • ¡Se agregaron las alertas!:

    Alertas de exito, advertencia y error para las distintas ocasiones.

  • Seccion de contacto:

    Ya sea para informar errores o simplemente recomendar una mejora.

  • Generador de contraseñas:

    Una herramienta muy util cuando se trata de elegir la mejor contraseña.

  • Opciones de las cuentas:

    Operaciones basicas, añadir, editar, eliminar y descargar cuentas.

  • Buscador de cuentas:

    Opcion dinamica para navegar por las cuentas si se tiene demasiadas.

  • Datos encriptados:

    Tanto tu contraseña de usuario como tus cuentas van a estar cifradas de extremo a extremo.


Implementacion de cuentas

La implementacion de cuentas nos permite manejar los diferentes usuarios para que cada uno tenga sus cuentas bien protegidas y organizadas. Nos registramos e inicamos sesion con los requerimientos establecidos para la seguridad del usuario. Y ya podremos manejar las diferentes herramientas que nos proporciona Sakura Secure Passwords.
